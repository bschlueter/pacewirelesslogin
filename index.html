pacewirelesslogin by nullcrux

An application to login without a browser to the PWN wireless network at Pace University.

The PWN wireless network at Pace University in New York usually requires a user to log in by accessing the login.pl script at pacewireless.pace.edu. The script and the application being developed automate this process so that a user may log into the network without using a browser.

Dependencies

Python, libcurl and pycurl. Is being developed on OSX 10.5, and periodically tested on linux. The basic script should work on any system so long as the dependencies are installed.

Install

Just execute and use. Simple.
